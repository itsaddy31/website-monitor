
Website Monitor Pro

FEATURES
- World latency ready (multi-region DB possible)
- Response time graphs
- 24h uptime %
- Add/remove sites UI
- JSON DB (can place in Google Drive folder)

USE GOOGLE DRIVE AS DATABASE
1. Install Google Drive for Desktop
2. Sync your folder locally
3. Note local path (example):
   C:\Users\You\My Drive\monitor\websites.json
4. Start server:
   set DB_PATH="C:\Users\You\My Drive\monitor\websites.json"
   npm start

RUN
npm install
npm start
http://localhost:3000
